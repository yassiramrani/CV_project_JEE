Projet JEE – Plateforme Intelligente de
Recrutement Basée sur l'IA
1. Contexte du Projet
Le recrutement est un processus long qui nécessite l'analyse de nombreuses candidatures et CV. Les
entreprises ont besoin d'outils permettant d'automatiser la présélection des candidats et de faciliter la
gestion des offres d'emploi.
Ce projet vise à développer une plateforme web de recrutement basée sur Java EE permettant aux
recruteurs de publier des offres d'emploi et aux candidats de postuler en ligne. Une fonctionnalité
d'intelligence artificielle sera intégrée afin d'analyser les CV et de calculer automatiquement un score de
compatibilité entre les candidats et les offres.

2. Objectifs du Projet
La plateforme devra permettre :
• Gestion des utilisateurs et authentification.
• Gestion des rôles (Administrateur, Recruteur, Candidat).
• Publication et gestion des offres d'emploi.
• Dépôt des candidatures.
• Téléversement des CV au format PDF.
• Analyse automatique des compétences présentes dans le CV.
• Calcul d'un score de compatibilité entre un candidat et une offre.
• Classement automatique des candidats.
• Consultation de statistiques simples via un tableau de bord.

Module 1 : Authentification
Fonctionnalités :
• Inscription.

Rôles :

• RECRUITER

Module 2 : Gestion des Offres
Le recruteur pourra :
• Créer une offre.
• Modifier une offre.
• Supprimer une offre.
• Consulter ses offres.
Le candidat pourra :
• Consulter les offres.
• Rechercher une offre.

Module 3 : Gestion des Candidatures
Le candidat pourra :
• Postuler à une offre.
Le recruteur pourra :
• Consulter les candidatures reçues.
• Consulter les scores des candidats.

Statuts possibles :

Module 4 : Gestion des CV
Fonctionnalités :
• Téléversement d'un CV PDF.
• Stockage du fichier.
• Consultation du CV.

Module 5 : Analyse des CV (IA)
Le système analysera automatiquement le CV afin d'extraire :
• Les compétences.
• Les expériences.
• Les certifications éventuelles.
Exemple :
CV :
Java, Spring Boot, Docker, PostgreSQL
Résultat :
Compétences détectées :
• Java
• Spring Boot
• Docker
• PostgreSQL

Module 6 : Matching et Classement Intelligent
Après analyse du CV, le système comparera les compétences du candidat avec celles demandées dans
l'offre.

• Compétences les plus recherchées.

recruteurs dans la sélection des candidats les plus pertinents.
